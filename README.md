# Neon Air Canvas 🎨

Advanced hand gesture-based spatial drawing interface utilizing real-time computer vision, the MediaPipe Tasks API, and OpenCV.

## Features ✨
- **DRAW**: Raise your index finger to draw with neon brush strokes and dynamic bloom.
- **ERASE**: Raise your index and middle fingers to trigger the circular eraser.
- **ADJUST**: Pinch your thumb and index fingers together to change the brush size in real-time.
- **CYCLE COLOR**: Clench your fist and open it to cycle between colors (Hot Pink, Cyan, Lime, Orange, Violet).
- **CLEAR ALL**: Open your palm (all 5 fingers) to instantly wipe the canvas.
- Beautiful Glassmorphic UI HUD.
- "Save Canvas" functionally mapped to the `S` key.

## Requirements
Ensure you have Python 3.10+ installed. Install the dependencies listed in `requirements.txt`:
```bash
pip install -r requirements.txt
```

## Running the App
Execute the main script:
```bash
python app.py
```
